It is the only functionality of the module.

Module and menu structure should be the same as the structure of PHP / Drupal document.
Experiments - /admin/ex/
- Data format and types - /admin/ex/data/
-- OAuth - /admin/ex/data/oauth
-- Soap - /admin/ex/data/soap
--- PHP Soap Client - /admin/ex/data/soap/php_soap
--- PHP Soap Server - /admin/ex/data/soap/php_server
